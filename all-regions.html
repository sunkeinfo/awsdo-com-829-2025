AWS 全球基础设施

亚马逊云科技 (AWS) 的全球云基础设施是最安全、扩展性和可靠性最高的云平台。以下列表汇总了 AWS 在全球范围内的主要地理区域和本地区域。

AWS 主要地理区域 (Regions)

每个 AWS 区域都是一个独立的地理位置，拥有多个相互隔离的可用区 (Availability Zones)。

AWS 本地区域 (Local Zones)
