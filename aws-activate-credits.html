AWS Activate 高額積分申請指南

最高可達 10 萬美元的 AWS 積分，助力初創企業快速成長

一、為什麼 AWS 願意給這麼多錢？

AWS 向初创企业提供高额积分（最高可达 10 万美元，约合 70 万人民币），主要是通过其 AWS Activate 计划实现的。这并不是针对所有企业的"普惠制"福利，而是一项旨在扶持早期高增长潜力企业的战略投资。

这背后的商业逻辑并非慈善，而是典型的"放长线钓大鱼"：

锁定未来的巨头：今天的初创公司可能是明天的 Netflix 或 Airbnb。AWS 通过初期免费供能，确保这些公司从第一行代码起就构建在 AWS 生态中。

培养用户习惯：云服务的迁移成本（Switching Cost）极高。一旦企业习惯了 AWS 的接口和架构，后期随着业务规模扩大，产生的付费收益将远超当初赠送的积分。

支持生态繁荣：通过积分吸引开发者和创新项目，能让 AWS 的技术栈（如 AI 模块 Bedrock、数据库 RDS 等）成为行业标准。

二、誰能拿到 10 萬美元積分？

10 万美元属于 Activate Portfolio（组合包）级别，主要面向已经获得机构支持的企业。

1. 核心申請門檧

機構背書（最關鍵）：

企业必须是由 AWS 认可的 Activate 合作伙伴（Activate Providers）支持的。这包括知名的风险投资机构（VC）、孵化器（如 Y Combinator）、加速器或共创空间。

企業階段：

通常要求是自筹资金或 B 轮以前（Pre-Series B）的初创公司。

成立年限：

通常要求公司成立时间在 10 年以内。

新用戶屬性：

公司之前未曾领取过同等或更高额度的积分。

官方證明：

需要有正式的公司網站、業務網域信箱，以及有效的 AWS 帳戶 ID。

2. 積分的分階段獲取

AWS 通常不会一次性直接给 10 万美元，而是采用"阶梯式"发放：

初创期：可能先给 $5,000 或 $10,000。

成长期：当公司业务扩大、原有积分快用完且有新的融资/机构背书时，可以申请"叠加"至最高 $100,000。

三、不同額度的企業類型對比

積分額度 對應專案 目標企業類型 申請核心要求
$1,000 Activate Founders 草根/個人創業者 只要有網站、未融資、成立10年內即可，申請難度最低。
$5,000 - $25,000 Activate Portfolio 天使輪/種子輪企業 需加入特定的孵化器或使用 Stripe Atlas 等服務入駐。
$100,000 Activate Portfolio 高增長/獲大牌VC投資企業 必須有 AWS 認可的高級合作夥伴提供的 Org ID（機構代碼）。

四、如何開始申請？

如果你認為你的企業符合條件，可以按以下步驟操作：

  1. 確認身份：查看你的投資人或所屬孵化器是否在 AWS 合作夥伴名單中。
  2. 獲取 Org ID：向你的投資機構索要專屬的 Organization ID。
  3. 提交申請：訪問 AWS Activate 官網，登入你的 AWS 帳戶並填寫企業資料。
  4. 等待審核：通常審核周期在 7-10 個工作日。

特別提醒：

積分通常有 1-2 年的有效期。如果你的業務還沒開始大規模消耗資源，建議不要過早申請大額積分，以免過期作廢。
