AWS Activate 高额积分申请指南

最高可达 10 万美元的 AWS 积分，助力初创企业快速成长

一、为什么 AWS 愿意给这么多钱？

AWS 向初创企业提供高额积分（最高可达 10 万美元，约合 70 万人民币），主要是通过其 AWS Activate 计划实现的。这并不是针对所有企业的"普惠制"福利，而是一项旨在扶持早期高增长潜力企业的战略投资。

这背后的商业逻辑并非慈善，而是典型的"放长线钓大鱼"：

锁定未来的巨头：今天的初创公司可能是明天的 Netflix 或 Airbnb。AWS 通过初期免费供能，确保这些公司从第一行代码起就构建在 AWS 生态中。

培养用户习惯：云服务的迁移成本（Switching Cost）极高。一旦企业习惯了 AWS 的接口和架构，后期随着业务规模扩大，产生的付费收益将远超当初赠送的积分。

支持生态繁荣：通过积分吸引开发者和创新项目，能让 AWS 的技术栈（如 AI 模块 Bedrock、数据库 RDS 等）成为行业标准。

二、谁能拿到 10 万美元积分？

10 万美元属于 Activate Portfolio（组合包）级别，主要面向已经获得机构支持的企业。

1. 核心申请门槛

机构背书（最关键）：

企业必须是由 AWS 认可的 Activate 合作伙伴（Activate Providers）支持的。这包括知名的风险投资机构（VC）、孵化器（如 Y Combinator）、加速器或共创空间。

企业阶段：

通常要求是自筹资金或 B 轮以前（Pre-Series B）的初创公司。

成立年限：

通常要求公司成立时间在 10 年以内。

新用户属性：

公司之前未曾领取过同等或更高额度的积分。

官方证明：

需要有正式的公司网站、业务域名邮箱，以及有效的 AWS 账户 ID。

2. 积分的分阶段获取

AWS 通常不会一次性直接给 10 万美元，而是采用"阶梯式"发放：

初创期：可能先给 $5,000 或 $10,000。

成长期：当公司业务扩大、原有积分快用完且有新的融资/机构背书时，可以申请"叠加"至最高 $100,000。

三、不同额度的企业类型对比

积分额度 对应项目 目标企业类型 申请核心要求
$1,000 Activate Founders 草根/个人创业者 只要有网站、未融资、成立10年内即可，申请难度最低。
$5,000 - $25,000 Activate Portfolio 天使轮/种子轮企业 需加入特定的孵化器或使用 Stripe Atlas 等服务入驻。
$100,000 Activate Portfolio 高增长/获大牌VC投资企业 必须有 AWS 认可的高级合作伙伴提供的 Org ID（机构代码）。

四、如何开始申请？

如果你认为你的企业符合条件，可以按以下步骤操作：

  1. 确认身份：查看你的投资人或所属孵化器是否在 AWS 合作伙伴名单中。
  2. 获取 Org ID：向你的投资机构索要专属的 Organization ID。
  3. 提交申请：访问 AWS Activate 官网，登录你的 AWS 账户并填写企业资料。
  4. 等待审核：通常审核周期在 7-10 个工作日。

特别提醒：

积分通常有 1-2 年的有效期。如果你的业务还没开始大规模消耗资源，建议不要过早申请大额积分，以免过期作废。
